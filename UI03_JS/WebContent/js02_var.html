변수의 사용

변수 선언 규칙
1. 대소문자 구분한다.
2. 영문, $, _로 시작해야 한다.
3. 영문, $, _, 숫자를 포함 할 수 있다.
4. 키워드나 예약어를 사용할 수 없다.
변수의 범위
1. 전역변수 : window 객체에 포함되는 변수로 다른 함수들에서 공용으로 사용할 수 있다.(값이 유지 됨)
2. 지역변수 : 함수나 객체 내부에서 선언되고 실행이 종료되면 사라짐
변수 선언 및 저장
1. var 키워드를 사용하여 변수 선언
2. 변수의 타입은 저장되는 값의 타입에 따라 결정된다.
타입의 종류
1. 문자(String)
2. 숫자(Number)
3. 논리(Boolean)
4. 널(Null)
5. 객체
6. 함수
